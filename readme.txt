== Description ==

This widget is a imple social link widget that shows user profile link with icon. Full features social widget with 16 site supported. Just enter your link and have fun. 
Easy to customize, just put your feed name and your widget ready to lunch.

You can use the built in installer and upgrader, or you can install the plugin manually.
1. Go to the menu 'Plugins' -> 'Install' and search for 'Simple Social Widget Widget'
2. Click 'install'.
